## Main Contents

- **Paper manuscript** – The full article (submitted to ACM AI Letters).  
- **Python cost model** – `cineinfini_cost_model_param.py` computes production costs for AI‑generated films (low/medium/high scenarios).  
- **Constants and parameters** – All constants with low/medium/high intervals, units, and references (`constants/Constants.md`).  
- **Bibliography** – Complete list of references (`bibliography/Bibliography.md`).  
- **Figures and prompts** – All figures and the exact AI generation prompts used (Gemini, Grok).  

---

## How to Use

1. **Reproduce cost estimates** – Run the Python script or use the Jupyter notebook (available in the paper folder).  
2. **Explore constants** – See `constants/Constants.md` for detailed tables.  
3. **Verify references** – All sources are listed in `bibliography/Bibliography.md`.  
4. **Re‑generate figures** – Use the provided prompts with Gemini, Grok, or any compatible image generator.
